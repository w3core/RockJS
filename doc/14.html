Templating and Precaching

Besides of embedding feature to separate reusable blocks such as templates RockJS supports inline templates.

Adding template via the script tag:

<script type="text/template" id="{id}">
  <!-- Template body -->
  This is the content of the template
</script>

For getting template inside of module or layout can be used $O.template(id[, DOM]) method (see details).

Also templates can be used for precaching of any URL. In other words, you can put content to the template and put his URL to the id attribute. Getting of content via $R.xhr by this URL will return content of this template instead of execution of AJAX HTTP request.

For example:

<script type="text/template" id="https://google.com/">
  <h1>Hello Google!</h1>
</script>

and

$R.xhr("https://google.com/", function(response){
  // Shows dialog with message "<h1>Hello Google!</h1>"
  alert(response);
});
